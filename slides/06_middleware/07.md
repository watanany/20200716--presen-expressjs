##  Middleware

| Middleware        | 効果                               |
|-------------------|------------------------------------|
| express-session   | セッション機能を提供する           |
| express-debug     | HTMLにデバッグ情報を出力する       |
| compression       | レスポンス圧縮する機能を提供する   |
| csurf             | CSRF対策                           |
| passport          | 認証機能を提供する                 |
| less-middleware   | LessをCSSに変換する                |
| coffee-middleware | JavaScriptをCoffeeScriptに変換する |
